Text Classifiction

  • Imported different genres of Gutenberg books written by different authors using NLTK library
  • pre-processed and Transformed the data using Bag of Words (BOW) and TF-IDF Model
  • Identified the required features to detect the author of a genre after giving a text
  • performed three different machine learning algorithms (Multinomial Naive Bayes, Decision Tree, Support Vector Machine and KNN) and discovered that SVM with TF-IDF is a champion model with 89% of accuracy
